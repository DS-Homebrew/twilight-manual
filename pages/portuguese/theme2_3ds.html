Sistema de Páginas
Os ícones no tema Nintendo 3DS estão divididos em páginas. com um máximo de 40 ícones por página. Podes mudar de páginas usando os botões & .
  • Carrega no botão na secção mais à direita para saltar para o último ícone na página.
  • Carrega no botão na secção mais à esquerda para saltar para o primeiro ícone na página.
Se os Botões de Ombro (Gatilhos) não funcionarem correctamente, use os botões SELECT + para fazer o mesmo que os botões de ombro (gatilhos).
