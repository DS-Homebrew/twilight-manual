Acerca de

O nds-bootstrap é uma aplicação homebrew usado pelo TWiLight Menu++ para carregar dumps de Game Cards e homebrew no modo DS através de um Cartão SD nos sistemas Nintendo DSi /3DS.

Há também uma versão para flashcards chamada B4DS, feita somente para executar homebrew em flashcards dedicados* ou flashcards com baixa compatibilidade. *Flashcards dedicados só para apps homebrew*

Compatibilidade

Para ver se o teu jogo é compatível com o nds-bootstrap, visite o site e veja se o seu jogo é compatível em:
cutt.ly/nds-compatibility

Para melhor compatibilidade com o B4DS, recomendamos que compre/use um DS Memory Expansion Pak. (só para Nintendo DS Original ou DS Lite)

Controlos
Pressione , , & para voltar ao TWiLight Menu++.
Pressione , , START e SELECT para reiniciar o jogo.
Pressione , , & para criar um dump da RAM.
Cheats
O nds-bootstrap pode aplicar cheats Action Replay através de uma base de dados usrcheat.dat, que precisa de ser colocado em sd:/_nds/TWiLightMenu/extras. Assim que a base de dados estiver lá, entre no menu de pré-configuração, e abra o menu de cheats ao usar o botão indicado.
Quer encontrar uma base de dados? Nós recomendamos usar a do DeadSkullzJr: cutt.ly/DSJCheats.
Ou se quiser, faça uma base de dados, sinta-se à vontade para usar a aplicação R4CEE num aparelho Windows (ou Wine se usares Linux ou macOS).
Esteja ciente que a implementação de cheats tipo-E do nds-bootstrap é instável e os códigos podem não funcionar. Isto não é uma falha na base de dados, e esperemos que este erro seja arranjado no nds-bootstrap muito em breve.
