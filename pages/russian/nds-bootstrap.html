О программе

nds-bootstrap — это homebrew приложение, используемое TWiLight Menu++ для загрузки картриджей и homebrew DS-mode с SD-карты Nintendo DSi/3DS.

Существует также версия для флешкарт B4DS, предназначенная для homebrew-only флешкарт и флешкарт с низкой совместимостью.

Совместимость

Чтобы узнать, совместима ли игра с nds-bootstrap, проверьте список совместимости:
cutt.ly/nds-compatibility

Для лучшей совместимости с B4DS, мы рекомендуем использовать DS Memory Expansion Pak.

Читы
nds-bootstrap может применять читы Action Replay через базу данных usrcheat.dat, которая должна находиться в sd:/_nds/TWiLightMenu/extras. После того, как база данных будет размещена, зайдите в меню настроек игры и откройте меню читов с помощью кнопки.
Нужно найти базу данных? Мы рекомендуем использовать DeadSkullzJr: cutt.ly/DSJCheats.
Или же, если вы хотите сделать это самостоятельно, не стесняйтесь использовать приложение R4CEE на устройстве Windows (или Wine, если вы используете Linux или macOS).
