Acerca de

nds-bootstrap es una aplicación de homebrew usada por TWiLight Menu++ para cargar volcados de cartucho y homebrew en modo DS desde la tarjeta SD Nintendo DSi/3DS.

También hay una versión para flashcarts titulado B4DS, destinada para flashcarts y flashcarts homebrew-only con baja compatibilidad.

Compatibilidad

Para ver si un juego es compatible con nds-bootstrap, revisa la lista de compatibilidad:
cutt.ly/nds-compatibility

Para la mejor compatibilidad con B4DS, recomendamos tener una expansión de memoria de DS.

Controles
Pulsa , , & para volver a TWiLight Menu++.
Pulsa , , START & SELECT para reiniciar el juego.
Hit , , & para crear una NAND "dump".
Trucos
nds-bootstrap puede aplicar trucos "Action Replay" a través de una base de datos usrcheat.dat, que debe colocarse en sd:/_nds/TWiLightMenu/extras. Una vez que la base de datos se coloque ahí, abre el menú de configuración del juego de interés y pulsa los botones que se muestran para abrir el menú de trucos.
Necesita encontrar una base de datos? Recomendamos usar el de DeadSkullzJr: cutt.ly/DSJCheats.
Alternativamente, si quieres hacer uno, siéntete libre de usar la aplicación R4CEE para Windows (o Wine si utilizas Linux o macOS).
