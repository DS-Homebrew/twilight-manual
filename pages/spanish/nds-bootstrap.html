Acerca de

nds-bootstrap es una aplicación homebrew usada por TWiLight Menu++ para ejecutar copias de seguridad de cartuchos y homebrew en Modo DS desde la tarjeta SD de la Nintendo DSi/3DS.

También hay una versión para flashcards titulada B4DS, destinada a flashcards solo para homebrew y flashcards con baja compatibilidad.

Compatibilidad

Para ver si un juego es compatible con nds-bootstrap, comprueba la lista de compatibilidad:
cutt.ly/nds-compatibility

Para tener mayor compatibilidad con B4DS, recomendamos tener un Memory Expansion Pak de DS.

Trucos
nds-bootstrap puede aplicar trucos de Action Replay a través de una base de datos usrcheat.dat, que debe colocarse en sd:/_nds/TWiLightMenu/extras. Una vez que la base de datos se encuentra ahí, abre el menú de configuración del juego en concreto y pulsa los botones que se muestran para abrir el menú de trucos.
¿Necesitas una base de datos? Recomendamos usar la de DeadSkullzJr: cutt.ly/DSJCheats.
Si quieres hacer una tú mismo, siéntete libre de usar la aplicación R4CEE para Windows (o Wine si usas Linux o macOS).
Ten en cuenta que la implementación de trucos de tipo E-type nds-bootstrap es inestable y como resultado, tu código puede no llegar a funcionar. Esto no es culpa de la base de datos, y esperamos solucionar este problema en nds-bootstrap lo antes posible.
