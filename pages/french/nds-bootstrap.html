À propos

nds-bootstrap est une application homebrew utilisée par le TWiLight Menu++ pour charger des dump de cartouches et des homebrew en mode DS à partir de la carte SD Nintendo DSi/3DS.

Il y a aussi une version pour les linkers sous le nom de B4DS, destinée uniquement aux linkers à base de homebrew-only et aux linkers à faible compatibilité.

Compatibilité

Pour voir si un jeu est compatible avec nds-bootstrap, vérifiez la liste de compatibilité suivante :
cutt.ly/nds-compatibility

Pour une meilleure compatibilité avec B4DS, nous vous recommandons de vous procurer un pack d'extension de mémoire Nintendo DS.

Contrôles
Appuyez sur , , & pour revenir sur TWiLight Menu++.
Appuyez sur , , START & SELECT pour redémarrer le jeu.
Appuyez sur , , & pour dump la NAND.
Triches
nds-bootstrap peut appliquer les triches Action Replay via une base de données usrcheat.dat qui devrait être placée à sd:/_nds/TWiLightMenu/extras. Une fois la base de données placée, entrez le menu de réglage de votre jeu et ouvrez le menu de triche en utilisant le bouton affiché.
Vous avez besoin de trouver une base de données ? Nous vous recommandons d'utiliser celui de DeadSkullzJr : cutt.ly/DSJCheats.
Alternativement, si vous souhaitez en faire un vous-même, n'hésitez pas à utiliser l'application R4CEE sur Windows (ou Wine si vous utilisez Linux ou macOS).
