Paramètres de l'interface graphique
Sur cette page, vous pouvez sélectionner le thème à utiliser et d'autres options qui affectent l'apparence du TWiLight Menu++.
Paramètres d'émulation/HB
Ici vous pouvez sélectionner les types roms à afficher et quel émulateur doit être utilisé.
Paramètres GBARunner2
Ces paramètres vous permettent de sélectionner la façon dont les jeux sont affichés et de les exécuter plus rapidement ou d'avoir une meilleure compatibilité.
Paramètres des jeux et des apps
Il y a les paramètres généraux pour la façon dont les jeux doivent être exécutés. La plupart d'entre eux varient selon la façon dont vous exécutez TWiLight Menu++, par exemple si vous utilisez le noyau de votre flashcard ou nds-bootstrap.
Divers. paramètres
Les options de cette page bascule sur la langue à utiliser, si vous voulez démarrer automatiquement le dernier jeu joué, si vous voulez afficher les écrans de démarrage au démarrage, entre autres.
Paramètres d'Unlaunch
Les paramètres ici vous permettent de changer le fonctionnement de Unlaunch, ils fonctionnent en patchant l'installateur Unlaunch, donc vous devrez le réinstaller pour que ces paramètres soient pris en compte.
Cette page n'apparaît que sur une console Nintendo DSi non lancée à partir d'un linker . Les utilisateurs d'une Flashcart et/ou d'une Nintendo 3DS ne verront pas cette page.
