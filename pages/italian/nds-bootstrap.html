Informazioni

nds-bootstrap è un'applicazione homebrew utilizzata da TWiLight Menu++ per caricare dump di cartucce e homebrew in modalità DS dalla scheda SD del Nintendo DSi/3DS.

C'è anche una versione per flashcart dal titolo B4DS, destinato a flashcart e flashcard only-homebrew con bassa compatibilità.

Compatibilità

Per vedere se un gioco è compatibile con nds-bootstrap, controlla la lista di compatibilità:
cutt.ly/nds-compatibility

Per avere una migliore compatibilità con B4DS, si consiglia di ottenere un DS Memory Expansion Pak.

Trucchi
nds-bootstrap può applicare i trucchi dell'Action Replayun tramite il database usrcheat.dat, che dovrebbe essere posizionato in sd:/_nds/TWiLightMenu/extras. Una volta che il database è stato posizionato lì, inserisci il menu di impostazione gioco per gioco e apri il menu dei trucchi usando il prompt dei pulsanti mostrato.
Hai bisogno di trovare un database? Si consiglia di utilizzare quello di DeadSkullzJr: cutt.ly/DSJCheats.
In alternativa, se vuoi crearne uno, sentitevi liberi di utilizzare l'applicazione R4CEE su un dispositivo Windows (o Wine se si utilizza Linux o macOS).
