Seitensystem
Menüeinträge im Nintendo DSi-Thema werden in Seiten aufgeteilt, mit maximal 40 Einträgen pro Seite. Du kannst durch die Seiten navigieren, indem du die & -Trigger benutzt.
  • Wenn du auf der Seite die ganz rechts ist drückst, gelangst du zum letzten Eintrag auf der Seite.
  • Wenn du auf der Seite die ganz links ist drückst, gelangst du zum ersten Eintrag auf der Seite.
Die untere Leiste repräsentiert die Position aller 40 Einträge auf einer Seite und so kannst du darauf tippen, um schnell einen bestimmten Ort auf einer Seite zu erreichen.
Wenn deine Trigger nicht funktionieren, funktionieren SELECT + als Ersatz.
Eigenes Bild am oberen Bildschirm
TWiLight Menu++ unterstützt das Anzeigen eigener Fotos auf dem oberen Bildschirm, genau wie das offizielle Nintendo DSi-Menü. Jedoch, anstatt sie aus der Nintendo DSi Kamera Anwendung zu beziehen, kannst du ein beliebiges PNG-Bild im folgenden Verzeichnis platzieren: sd:/_nds/TWiLightMenu/dsimenu/photos
  • Maximale Breite: 208 Pixel
  • Maximale Höhe: 156 Pixel
Wenn die Bildgröße kleiner als die Maximalgröße ist, werden die schwarzen Ränder dein Bild umgeben.
SELECT-Menü
Das Drücken von SELECT im Nintendo DSi-Thema öffnet standardmäßig das Klassische DS-Menü. In den TWiLight Menu++ Einstellungen kannst du es jedoch so ändern, dass es das SELECT-Menü öffnet, ein Miniaturmenü, das im Thema selbst eingebettet ist. Hier sind die Menüoptionen für das SELECT-Menü.
  • Home-Menü: Auf den Nintendo DSi- und Nintendo 3DS-Konsolen wird diese Option zur Rückkehr ins Home-Menü angezeigt.
  • Einstellungen: Auswahl öffnet ein Menü zum Umstellen von Einstellungen für TWiLight Menu++ und dessen Launcher.
  • Spielkarten-Optionen: Auf einem original-DS oder -DS Lite kannst du dein Slot-2-Gerät von hier aus starten. Auf einem Nintendo DSi und Nintendo 3DS auf dem TWiLightMenu++ von der SD-Karte läuft, kannst du entweder deine Slot-1-Karte starten oder auf bestimmten Flashkarten, umstellen welche SD-Karte TWiLight Menu++ navigiert.
  • Anleitung: Dies startet die Informationsdatenbank von TWiLight Menu++. Es ist das, was du gerade ansiehst :P
