קובץ BIOS
GBARunner2 can use a dump of the official GBA BIOS to improve compatibility. The file must be named bios.bin and can be in the root of the SD card, sd:/gba or sd:/_gba.
Make sure your GBA BIOS dump matches with the following CRC32 hash: 81977335.
You can check this by visiting the following website: simplycalc.com/crc32-file.php
